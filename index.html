Git 新手操作筆記：從本機專案上傳到 GitHub

1. 初始化本地 Git 儲存庫

git init

2. 設定 Git 使用者資訊（只需設定一次）

git config --global user.name "你的名字"
git config --global user.email "你的Email"

3. 把檔案加入 Git 追蹤

git add .

4. 建立第一次 commit

git commit -m "建立一頁式網頁"

5. 將 GitHub Repo 加入遠端來源

git remote add origin https://github.com/你的帳號/你的-repo.git

6. 如果你的分支名稱是 `master`，改成 `main`（可選）

git branch -m main

7. 推送到 GitHub

git push -u origin main

之後的更新流程：

每次更新網頁後，只需重複這三個指令：
git add .
git commit -m "這次做了什麼改動"
git push

常見錯誤解決：

錯誤訊息 解法
'git' 不是內部或外部指令 沒有安裝 Git，先從 git-scm.com 下載並安裝
src refspec main does not match any 本地分支名稱不是 main，改成 main 或推送 master
Author identity unknown 沒有設定 user.name 或 user.email，照第 2 步設定
